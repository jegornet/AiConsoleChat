СТРУКТУРА ПРОЕКТА
main.py - Точка входа приложения, координирует работу всех компонентов и управляет жизненным циклом.
config.py - Файл конфигурации с константами приложения: модель Claude, токены, режимы ввода.
mcp_connection.py - Класс для работы с MCP серверами через stdio, управляет инструментами и соединением.
claude_chat.py - Класс для взаимодействия с API Anthropic Claude, обрабатывает запросы и инструменты.
chat_interface.py - Класс пользовательского интерфейса с поддержкой многострочного ввода и защитой от пустых запросов.
slash_commands.py - Класс для обработки слеш-команд: /quit, /max_tokens, /multiline.
mcp_config_loader.py - Класс для загрузки и валидации конфигурации MCP серверов из mcp.json.
mcp_server.py - MCP сервер файловых операций с инструментами list_files, read_file, write_file, delete_file.
